# PROVIO

AI-assisted pantry operations for intake, transfers, and audit checkpoints.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
